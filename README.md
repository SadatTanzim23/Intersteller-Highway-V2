# INTERSTELLER HIGHWAY <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Telegram-Animated-Emojis/main/Travel%20and%20Places/Rocket.webp" alt="Rocket" width="40" height="40"/>

A program created to optimize the best trade routes in space. This is a platform built specifically for smart, scalable routes, 
a logistics planning platform that will serve as the foundation of a safer, more reliable space trade infrastructure.

## <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Objects/Bookmark%20Tabs.png" alt="Bookmark Tabs" width="40" height="40" /> Project Overview 

As humanity expands its reach out into the stars, the Golden Age of Space Commerce is fully underway. Interplanetary trade has transformed 
the way we live, work, and connect; space commerce has become the backbone of this new modern economy. To properly support this rapidly 
growing network; a new initiative has emerged: A spacefaring logistics platform that can help merchants, cargo fleets, and independent 
traders plan the safest, and most efficient routes between planets, and stations. The goal is to build a smart, scalable route planning 
logistics platform that will serve as the foundation of a safer, and more reliable space trade infrastructure.

## <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Activities/Bullseye.png" alt="Bullseye" width="40" height="40" /> Purpose

The purpose of this system is to simulate an interplanetary trading economy where rockets with varying cargo capacities, speeds, and fuel 
efficiencies transport goods between planets under realistic constraints.
#### The system enables: 
* Route optimization (fastest vs most fuel-efficient)
* Cargo capacity management
* Fuel consumption modeling
* Time-based trade decisions
* Dynamic supply and demand economics
* Strategic fleet planning

## <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Telegram-Animated-Emojis/main/Objects/Keyboard.webp" alt="Keyboard" width="35" height="35" /> Getting Started
#### To run the program locally through terminal:
1. Clone the repository:
   ```bash
   git clone https://github.com/SadatTanzim23/Intersteller-Highway-V2.git
   ```
2. Navigate to the project folder:
   ```bash
   cd Intersteller-Highway-v2
   ```
3. Activate Python virtual environment:
   ```bash
   source .venv/bin/activate
   ```
4. Run the program:
   ```bash
   python game.py
   ```
